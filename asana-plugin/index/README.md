# Index Layer

The workspace knowledge base (players, projects, fields, options) has been extracted into the standalone `anasa-context` skill for reuse across all skills and platform variants.

**Load from:** `anasa-context/SKILL.md` and its `index/` subdirectory.

The index files in this directory are retained as local copies for offline/packaging use. Canonical source is `anasa-context/`.

Future: these files will be generated by `GET /snapshot?format=markdown` from the Anasa API. See `ANASA-PRD.md`.
